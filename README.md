# souls

create actual living souls inside your computer!

requires:
- python 3
- numpy
- pyglet
- currently a shitload of cpu time

current state:
- renders souls at 30 fps
- reads simple rules
- mostly terrible so far

syntax:
- tokens:
  - r, g, b, k, w = colors (red, green, blue, black, white)
  - ^, v, <, > = directions
